# 🚗 PitStop – Vehicle Service Center Management System

PitStop is a Python-based desktop application designed to manage the daily operations of a vehicle service center. It provides a complete system for handling customers, vehicles, job assignments, mechanics, inventory, billing, and service tracking in one centralized platform.

This project was developed as a **first-year, first-semester university project** to demonstrate real-world software development using Python and MySQL.

---

## 📌 Project Purpose

The purpose of PitStop is to digitize and simplify the workflow of a vehicle service center by reducing manual paperwork and improving efficiency in managing operations such as job assignments, service tracking, and billing.

---

## ⚙️ Features

- Customer management system  
- Vehicle management system  
- User management system  
- Inventory management system  
- Job creation and assignment system  
- Mechanic job allocation  
- Job progress tracking  
- Job history tracking  
- Billing and invoice generation  
- Role-based login system (Admin / Mechanic)  
- Admin dashboard  
- Mechanic dashboard  
- Job status tracking  

---

## 💻 Tech Stack

---

## 🚀 How to Run

Run the application by opening:

**Pitstop_V6.exe**

---

## 🧠 System Overview

PitStop uses a role-based architecture:

- **Admin**: Manages customers, vehicles, mechanics, inventory, job assignments, and billing  
- **Mechanic**: Views assigned jobs, updates progress, and tracks job history  

This ensures smooth communication and structured workflow inside the service center.

---

## 🗄️ Database

The system uses MySQL to manage all data, including:

- Customers  
- Vehicles  
- Jobs  
- Mechanics  
- Inventory  
- Billing records
